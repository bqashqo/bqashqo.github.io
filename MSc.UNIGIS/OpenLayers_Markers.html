Openlayers Simple map - By: Bassam QASHQO
This map was amended based on the OpenLayers Simple map provided. I used the OpenLayers Marker Array which allows us to loop and add markers, as a vector layer, to the map according to an array of coordinates.

I had some following changes on the code such as: change the map id, add the source of the marker, add header, add description, add icon to the page, define a proper style for each (the map, the header and the description).
